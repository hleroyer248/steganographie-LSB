Programme stéganographique 

Genre : Transformateur d'image

Plateforme : Windows 

Auteur : Hugo Leroyer/Lino Lomboni 

Ce Programme permet d'include un message caché dans une image, l'enregistrer et même lire les messsages secrets des images, en utilisant la Stéganographie.

Installation Prérequis :
- Windows 10 ou supérieur
- Compilateur C++ compatible C++14 (Visual Studio recommandé)
  
Configuration des assets : Aucun asset nécessaire pour ce programme
Commandes : zoom possible avec la molette de la souris.

Interface : il y a 3 bouton :
- bouton fichier, pour ouvrir une image, l'enregistrer ou encore quitter.
- bouton Stéganographie, pour insérer un message caché, ou le lire si l'image en comprte.
- bouton affichage, pour recentrer l'image et la remettre à sa taille initiale.
